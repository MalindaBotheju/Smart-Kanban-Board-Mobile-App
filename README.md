A fully functional, cloud-connected Kanban board Android application designed to help users manage their daily tasks efficiently. Built using modern Android architecture and Firebase.

## ✨ Features
* **User Authentication:** Secure email/password login and account creation.
* **Kanban Workflow:** Easily create, update, and organize tasks into "TODO", "IN PROGRESS", and "DONE" columns.
* **Cloud Sync:** Real-time database updates using Firebase Firestore.
* **Offline Persistence:** Tasks can be viewed and updated without internet access, automatically syncing once the connection is restored.
* **Optimized UI:** Smooth scrolling and efficient memory management using `RecyclerView`.

## 🛠️ Tech Stack
* **Language:** Java
* **UI/Layout:** XML, Material Design Guidelines
* **Database:** Firebase Cloud Firestore
* **Authentication:** Firebase Auth
* **Architecture:** ViewPager2, Fragments, RecyclerView, Custom Adapters
